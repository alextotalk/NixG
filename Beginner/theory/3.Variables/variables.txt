Переменная - это именованное место хранения какого-то типа данных


- знать, как объявляются переменные?

Переменные в Go создаются с помощью ключевого слова var, за которым следуют имя переменной (x),
тип (string) и присваиваемое значение (Hello World). Последний шаг не обязателен
    var x string
    x = "Hello World"
    fmt.Println(x)


- допустимые значения в именовании переменных?

Имена должны начинаться с буквы и могут содержать буквы, цифры и знак _ (знак подчеркивания).


- знать, что такое инициализация переменной с автоопределением типа, и уметь применять на практике?

Места, где может использоваться переменная x, называются областью видимости переменной.Согласно спецификации
«в Go область видимости ограничена блоками». В основном это значит, что переменные
существуют только внутри текущих фигурных скобок { } (в блоке), включая все вложенные скобки (блоки).
Если мы хотим присвоить значение переменной при её создании, то можем использовать сокращенную запись:
x := "Hello World"

- область видимости?

теперь другие функции имеют доступ к этой переменной:
var x string = "Hello World"

func main() {
    fmt.Println(x)
}

func f() {
    fmt.Println(x)
}


Если вы попробуете выполнить эту программу, то получите ошибку:
func main() {
    var x string = "Hello World"
    fmt.Println(x)
}

func f() {
    fmt.Println(x)
}
